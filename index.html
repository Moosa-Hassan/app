Tools to help in designing Websites - Front End Frameworks. When building websites, great emphasis is paid to the design and layout of the website as this is what gives the first impression to the potential users. The process of designing this layout/design is called the Front End. Now to accelerate the process of developing Front End, there are frameworks which help a developer to build websites much faster and more efficiently. These frameworks are basically compilation of ready made code to be integrated into your projects. Below are a few examples of such beginner friendly frameworks: Bootstrap: It provides a straightforward way to design responsive components Helps make visually appealing websites Easy to create websites without deep knowledge of CSS.
